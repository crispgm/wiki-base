Personal wiki as a knowledge base. The wiki language is either in English or in Chinese.

Dev

Bootstrap

Clone, and then set vimwiki path to it.

Publish

make init
make all
make clean

Templates

Crisp Wiki Template

Filters

  • breadcrumb-menu: generate breadcrumb menu
  • links-to-html: convert inner links from .md to .html
  • promote-headers: promote header as title
