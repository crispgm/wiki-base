Moving

  • l right
  • w word forward

Scrolling

Searching/Replacing

  • # search currenct word
  • / search input word
  • ? search input word from bottom
  • /<xxx/> match whole word of “xxx”
  • %/s/src/dst replace “src” with “dst”
  • %/s/src/dst/g replace all “src” with “dst”
  • %/s/src/dst/c replace “src” with “dst”, with confirmation
